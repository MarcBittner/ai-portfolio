Help · agent-factory

What it is

Build a tool-using agent from a declarative spec — system prompt, tool allowlist, planner, model tier, step budget, and guardrails — then run a task and watch the thought → action → observation trace. The deterministic core runs fully offline; a free model sharpens planning and the final answer.

How to use

  1. Pick a template — choose one of the cards (assistant, calculator, researcher, analyst) to seed the spec.
  2. Enter a task — type into the Task box (or click a sample), then Run agent.
  3. Go deep — open the gear to edit the spec: prompt, tools, planner, routing mode, model override, max steps, temperature, answer style, and guardrails.
  4. Run your spec — tick Run with this custom spec in the drawer footer to run the edited spec instead of the template.
