Help · agent-sandbox

What it is

A ReAct-style agent over a set of safe, deterministic tools (math, unit conversion, date math, project lookup). It reasons, calls one tool, observes the result, and repeats — chaining results across steps. The planner is the deterministic rule planner by default; an LLM planner plugs in behind the same interface and routes through a multi-provider chain.

How to use

  1. Enter a query — type into the box or click a sample chip, then Run agent (or press Enter).
  2. Watch the trace — each step shows the agent's thought, the tool call it makes, and the observation it gets back; the final Answer card sums it up.
  3. Choose a planner — toggle LLM planner off to force the rule planner; if the LLM is unavailable the run silently falls back to rules.
  4. Pick a route — open Settings ⚙ to set the routing mode + model override and see provider status.
