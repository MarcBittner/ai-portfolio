Help · attack-surface

What it is

An attack-surface report that turns an exposure scan into governance evidence: discovered assets → findings → a SOC 2 / ISO 27001 control crosswalk → a posture grade → a board-ready executive risk narrative, with a remediation diff that shows the posture lift.

How to use

  1. Run a scan — click Run scan on the Posture panel for the control-mapped report (or set live mode in Settings for passive CT recon).
  2. Read findings & controls — Findings → controls maps each finding to controls; Control coverage shows per-control status.
  3. Exec narrative — the Executive risk narrative panel generates a board-ready summary + prioritized remediations via the routing chain.
  4. Remediation diff — Remediation diff shows the before/after posture lift from fixing the criticals.
