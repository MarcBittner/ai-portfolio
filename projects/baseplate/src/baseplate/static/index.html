What it is
A platform paved-road console: a self-service LLM service scaffolder that turns a
      free-text description into a ServiceSpec, then generates Terraform / k8s / golden CI / SLO files and onboards the
      service to a catalog — plus an example ingest workload with a data-quality SLI and an SLO view.

    
How to use
    
      
  1. Scaffold a service — describe it in Scaffold a service (or click an example), choose whether to
        onboard, and click Generate paved-road files.
    2. 
      
  2. View generated files — switch tabs in Generated files to read each templated paved-road artifact.
    3. 
      
  3. The catalog — onboarded services appear in Service catalog with how they were routed.
    4. 
      
  4. Run ingest — click Run ingest to score the synthetic rate file's data-quality SLI.
    5. 
      
  5. SLOs — the SLOs panel shows objectives, error budgets, and the live data-quality SLI.
    6.
