Help · doc-extract

What it is

Schema-driven structured extraction: a deterministic, label-anchored extractor pulls fields with per-field confidence, type validation, and provenance (the exact span each value came from). Optional LLM fill recovers fields the regex core misses, routed through a multi-provider chain.

How to use

  1. Pick a schema — choose an extraction schema (invoice, resume, contact…) at the top.
  2. Paste a document — drop source text into Document, or click Load sample.
  3. Extract — click Extract; matched spans highlight in Provenance.
  4. Review — read the Fields table (value, confidence bar, valid badge, method) and copy the JSON.
