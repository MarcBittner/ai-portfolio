Help · evalkit

What it is

An offline-first LLM evaluation toolkit: score model predictions against references across layered deterministic metrics, add an optional LLM-judge grade, and gate a release on a regression threshold. Deterministic metrics need no model or network; the judge routes through a multi-provider chain that always ends at a deterministic fallback.

How to use

  1. Paste items — one per line as prediction ||| reference in the Items panel.
  2. Pick metrics — check the metrics to score; add the llm_judge metric for an LLM grade.
  3. Set thresholds — type a minimum next to a metric to arm the regression gate (blank = no gate).
  4. Evaluate — see aggregate bars, the PASS/FAIL gate badge, and the per-item score table.
