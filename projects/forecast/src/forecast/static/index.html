Help · forecast

What it is

Classic-ML time-series forecasting and anomaly detection. Seven deterministic methods (naive, mean, linear trend, SES, Holt, seasonal-naive, Holt-Winters) plus auto, which backtests and picks the best. Every forecast ships a 95% prediction band and a z-score anomaly scan. The core is pure Python with no model — the optional LLM only writes a plain-language summary; the numbers stay deterministic.

How to use

  1. Enter a series — paste numbers (commas, spaces, or newlines) or load a CSV; the last numeric column is used.
  2. Try a sample — click trend, seasonal, or spike to autofill and run.
  3. Pick a method & horizon — leave method on auto to let the backtest choose.
  4. Forecast — the chart shows history (solid), forecast (dashed), the 95% band, and anomalies (red dots); the meta row reports holdout/rolling MAE.
