What it is
A document redaction pipeline: OCR turns an image into positioned text tokens,
      PII detection finds sensitive spans (deterministic regex + optional LLM NER), and
      box-level redaction maps each PII span back to the OCR tokens it covers and blacks out their boxes —
      so the redaction is visual, not just textual.
