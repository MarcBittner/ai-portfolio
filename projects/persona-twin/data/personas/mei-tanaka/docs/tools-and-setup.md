# Tools, setup, and the rituals around them

The engine is Godot, switched from Unity after *Attic Light* shipped —
partly licensing anxiety, partly because I wanted the rewrite excuse.
(I always want the rewrite excuse. See: everything.)

The rest of the stack:
- Aseprite for pixel art; my palette is 24 colors and has not changed
  in six years
- Audacity plus a $90 field recorder for sound — the foghorn in the
  demo is a ferry recorded from my kitchen window, pitched down
- Obsidian vault for design notes, 1,400 notes, organization
  aspirational
- Git with commit messages that range from "salience v3" to "I'm sorry"

I work nights, roughly 22:00 to 03:00. Mornings are for email and
guilt. The night work isn't discipline — it's just when the
second-guessing gets quiet enough to build things.

One rule that survived both projects: every Friday I cut a build, even
if the only change is a typo. A project that always builds is a
project that's still alive.
