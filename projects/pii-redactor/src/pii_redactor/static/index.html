Help · pii-redactor

What it is

Deterministic PII detection & redaction. A regex+checksum core (Luhn-validated cards, mod-97 IBANs) finds structured identifiers with no model; an optional LLM named-entity pass adds names, orgs, and places. Detection may use a model, but redaction is always deterministic.

How to use

  1. Paste text into Input (or click Load sample).
  2. Detect & redact — see PII highlighted by type with per-type counts.
  3. Switch redaction style — token / label / mask / partial / hash.
  4. Copy the redacted output.
