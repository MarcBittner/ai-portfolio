Help · promptguard

What it is

A deterministic LLM firewall. It scans prompts (input) for injection & jailbreak attempts and model responses (output) for secret & PII leakage, returning allow / flag / block with per-finding detections. An optional LLM classifier adds a semantic check for paraphrased injection on the input. It never echoes a secret it catches.

How to use

  1. Paste text — a prompt or a model response into the scan console.
  2. Pick a direction — input, output, or both; toggle the LLM classifier.
  3. Scan — read the verdict, the highlighted findings, and the detections table by category.
  4. Try an example — the sample chips load injection, jailbreak, secret-leak, PII, and benign cases.
