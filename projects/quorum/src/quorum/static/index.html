What it is

A vendor-neutral multi-agent orchestrator with governance baked in: declarative workflow specs run through one governed engine, every step records its routing tier, PII is redacted before the model and the audit, and each run produces a tamper-evident, hash-chained audit.

How to use

  1. Run a workflow — pick a contract (or paste text) and a routing mode in Run a governed workflow, then Review contract.
  2. View the agent trace — watch the Agent pipeline DAG light up and read the Per-step trace (provider · model · latency · cost).
  3. See the audit — the Tamper-evident audit chain shows the hash-chained, PII-redacted steps and the verify result.
  4. Try the pattern — the same engine runs policy-qa from a different spec in Replicable pattern.
