LLM-assisted mapping — an unknown-format file offline

The three adapters are hand-written per known shape. Paste a 4th file in an unknown format (JSON array, or CSV/pipe/TSV with a header) — the model maps each column to the canonical schema (code · code_type · payer · plan · rate · hospital · description), then it's applied deterministically to ingest. Routes Anthropic/OpenAI → Ollama → OpenRouter → a deterministic synonym matcher, so it works with zero keys.
