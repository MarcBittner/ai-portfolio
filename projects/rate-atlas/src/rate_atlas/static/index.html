Help · rate-atlas

What it is

A negotiated-rate comparison tool: it normalizes inconsistent price-transparency files (CMS nested JSON, flat JSON, pipe CSV — and unknown formats via an LLM-assisted mapper) into one canonical model, then compares a procedure's rate across payers and hospitals and flags statistical outliers.

How to use

  1. Compare a procedure — pick a code in the top panel and click Compare to see lowest / median / highest / spread and per-payer rates.
  2. Scan outliers — the Rate outliers panel flags rates by |z|-score; tune the threshold in Settings.
  3. Review sources — Sources shows each file's detected shape mapped to canonical.
  4. Map an unknown file — paste a 4th file (or load the sample) into LLM-assisted mapping and click Map & ingest; the model proposes a column → canonical mapping, applied deterministically.
